## [0.0.1] - 2025-06-15

### Added
- Initial release of newrelic-publisher
- Vite plugin integration for automatic sourcemap upload to New Relic
- Automatic sourcemap discovery from build directory
- Environment variable support for configuration
- Dry run mode for testing
- Sourcemap cleanup after successful upload
- Comprehensive error handling and logging
- TypeScript support with full type definitions
- Manual upload API via publishSourcemapAsync utility
- Duplicate detection for already-published sourcemaps
- Flexible configuration options
- GitHub Actions CI/CD pipeline
- Complete documentation and examples
